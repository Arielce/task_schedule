# task_schedule

## 项目介绍

基于DAG图的调度DEMO，基于图数据结构与拓扑序列理论，实现一个任务调度类。

## 输入

```
    /*
     *
     *          J  O  B 1
     *          /   \   \
     *        V      V    V
     *     JOB2    JOB3   JOB5
     *         \    /
     *         V   V
     *         JOB4
     */
```
